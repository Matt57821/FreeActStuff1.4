Installation Guides

  1. Download the .mcaddon file from the add-on page
  2. Open Minecraft Bedrock Edition on your device
  3. Double-click the downloaded .mcaddon file
  4. Minecraft will open automatically and import the add-on
  5. Create a new world or edit an existing world
  6. Go to Behavior Packs and activate the mod
  7. Start playing with your new mod!
  1. Download the .mcpack file
  2. Double-click to open with Minecraft
  3. The texture pack will be imported automatically
  4. Go to Settings > Global Resources
  5. Find your texture pack and activate it
  6. Apply changes and enjoy your new textures!
  1. Download the .mcworld file
  2. Double-click the file to open with Minecraft
  3. The world will be imported automatically
  4. You'll see the world in your worlds list
  5. Click on the world to start playing
  6. Explore and have fun!
